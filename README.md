# 🇹🇷 Turkey Economic Dashboard

An interactive economic dashboard built with Python and Streamlit using real-world data.

## Features

- Interactive indicator selection
- Inflation and unemployment data for Turkey
- Data table and trend chart
- Live web app deployment

## Data Source

World Bank data files used for:
- Inflation
- Unemployment

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib

## Run Locally

```bash
git clone https://github.com/ipekpala/economic-dashboard.git
cd economic-dashboard
python3 -m virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
